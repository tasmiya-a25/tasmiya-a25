## 👋 About Me

I'm an AI/ML Engineer and Full-Stack Developer focused on building **production-grade systems** — not just notebooks. My work spans multi-agent orchestration, retrieval-augmented generation, and scalable backend architecture, with an emphasis on the kind of engineering rigor that survives contact with real users.

- 🔭 Currently building **enterprise-grade AI platforms** with multi-agent orchestration (LangGraph) and hybrid retrieval systems
- 🌱 Deepening expertise in **distributed systems**, **vector search at scale**, and **LLM evaluation frameworks**
- 💬 Ask me about **FastAPI architecture, RAG pipelines, async Python, or Docker/Kubernetes deployments**

### 🤖 Enterprise Multi-Agent AI Platform
Multi-agent orchestration platform built on **FastAPI + LangGraph**, handling complex task delegation across specialized agents with WebSocket streaming and async job processing.

- LangGraph-based agent orchestration with state propagation
- Real-time streaming via WebSockets + Celery for background jobs
- PostgreSQL + Redis + Qdrant for persistence, caching, and vector search
